# 📊 Recruitment Dashboard (Power BI)

## 📌 Overview

The **Recruitment Dashboard** is an interactive Power BI report designed to analyze hiring trends, workforce distribution, and recruitment channel performance. It helps HR professionals and business leaders make data-driven decisions by providing clear insights into employee hiring patterns, job roles, and diversity metrics.

---

## 🎯 Objectives

* Analyze yearly hiring trends
* Evaluate effectiveness of recruitment sources
* Understand workforce composition by roles
* Track gender diversity
* Monitor key hiring KPIs

---

## 📊 Key Metrics

* **👥 Total Employees:** 311
* **💼 Number of Positions:** 30
* **🌐 Recruitment Sources:** 9
* **💰 Maximum Salary:** 250K

---

## 📈 Dashboard Components

### 📅 Employees Hired Yearly

* Funnel chart showing hiring distribution across years
* Identifies peak recruitment periods and hiring fluctuations

---

### 📢 Recruitment Source Analysis

* Displays employee count based on hiring platforms:

  * LinkedIn
  * Indeed
  * Google Search
  * Employee Referral
  * Job Fair
  * Company Website
* Helps determine the most effective recruitment channels

---

### 🧑‍💼 Position Distribution

* Pie chart representing number of employees across different roles:

  * Production Technician I & II
  * Software Engineer
  * Data Analyst
  * IT Support
  * Sales & Management roles
* Useful for workforce structure analysis

---

### ⚖️ Gender Ratio (Male vs Female)

* Donut chart showing gender distribution
* Supports diversity and inclusion insights

---

### 📦 KPI Cards

* Quick summary of:

  * Total employees
  * Positions available
  * Recruitment sources
  * Maximum salary offered

---

## 🛠️ Tools & Technologies

* **Power BI Desktop** – Data visualization and dashboard development
* **Data Modeling** – Relationships, calculated measures, and transformations

## 🚀 Getting Started

### 🔧 Prerequisites

* Install **Power BI Desktop**

### ▶️ Steps to Run

1. Download or clone this repository
2. Open the `.pbix` file in Power BI Desktop
3. Refresh the dataset if required
4. Interact with visuals to explore insights
