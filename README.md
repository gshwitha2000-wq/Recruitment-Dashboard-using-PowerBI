# 📊 Recruitment Dashboard (Power BI)

## 📌 Overview

This Recruitment Dashboard provides a comprehensive analysis of hiring trends, recruitment sources, employee distribution, and salary insights within an organization. Built using Power BI, the dashboard enables HR teams and decision-makers to monitor recruitment performance and optimize hiring strategies.

---

## 📊 Key Metrics

* **Total Employees:** 311
* **Number of Positions:** 30
* **Recruitment Sources:** 9
* **Maximum Salary Offered:** 250K

---

## 📈 Dashboard Features

### 📅 Employees Hired Yearly

* Visual representation of hiring trends over the years
* Helps identify peak hiring periods and recruitment slowdowns

---

### 📌 Recruitment Source Analysis

* Displays employee count by recruitment channels such as:

  * LinkedIn
  * Indeed
  * Google Search
  * Employee Referral
  * Job Fairs
* Useful for evaluating the effectiveness of each hiring source

---

### 🧑‍💼 Position Distribution

* Pie chart showing the number of hires per job role
* Includes roles like:

  * Production Technician
  * Software Engineer
  * Data Analyst
  * IT Support
* Helps understand workforce composition

---

### ⚖️ Gender Distribution

* Donut chart representing male vs female employee ratio
* Supports diversity and inclusion analysis

---

### 📦 Summary Cards

* Quick overview of:

  * Total employees
  * Total positions
  * Recruitment sources
  * Maximum salary

---

## 🎯 Use Cases

* HR analytics and workforce planning
* Recruitment strategy optimization
* Diversity tracking and reporting
* Identifying high-performing hiring channels
* Salary benchmarking and budget planning

## 🛠️ Tools & Technologies

* **Power BI** – Dashboard creation and data visualization
* **Data Processing:** Data cleaning, transformation, and modeling within Power BI

## 🚀 How to Use

1. Open the `.pbix` file in Power BI Desktop
2. Interact with visuals to explore recruitment trends
3. Analyze hiring sources and employee distribution
4. Use insights for HR decision-making

## 📌 Key Insights

* Identify which recruitment platforms bring the most candidates
* Understand hiring trends across different years
* Analyze workforce diversity
